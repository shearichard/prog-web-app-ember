## Deploying to Heroku
Here's a selection of commands used to interact with Heroku. 

### Deploy latest version

```
git push heroku master
```

### Run bash shell on Heroku instance

```
heroku run bash
```

### See latest log entries of the Heroku instance

```
heroku logs --tail
```
